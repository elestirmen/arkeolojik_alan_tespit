# Workspace

Repo-ici veri, model, cache ve cikti dosyalari `workspace/` altinda tutulur.

Onerilen alt dizinler:

- `workspace/on_veri/`
- `workspace/training_data/`
- `workspace/training_data_classification/`
- `workspace/checkpoints/`
- `workspace/cache/`
- `workspace/ciktilar/`
- `workspace/assets/`

Bu dizinin icerigi `.gitignore` ile repo disi tutulur; yalnizca bu README izlenir.
